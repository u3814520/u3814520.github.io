line Bot 大樂透兌獎-Azure圖像辨識

By 巧庭_Ann 14 September 2021

進入滿滿程式碼的世界兩個月，

最近又有一個小作品啦~~

距離大學畢業後，再也沒有團隊合作做作業了，

這是一個Azure的小作品

談談這個作品的故事吧~

起初大家不知道要做什麼題目，

在討論聊天的過程中，一位同學說了最近大樂透頭獎累積到27億，上課前去買了一張，

這時另一位同學說:我們就來做大樂透對獎結合Azure圖像辨識，並在line bot呈現吧!!!

架構(運用Azure課程中所教導的技術及知識):

  • Microsoft Azure - Computer vision影像辨識功能

  • Line Developer - 以Line Bot呈現至使用者服務端

  • Python - Flask - 作為服務後端，並以python語法串聯

  • Ngrok - 簡易可生成https服務

中間遇到的困難是:

  • 圖像辨識有時會有小差錯，須調整影像色調測試。

  • 圖像若拍攝稍有不清時，可能造成讀取辨識失敗。

  • Ngrok免費帳戶有流量限制，圖像不能傳送太快，否則會造成Error。

經過兩個禮拜時間，成品出來啦~~

一點一點對程式碼的進步與了解，
