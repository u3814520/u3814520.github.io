[分享]Github免費架設個人網站

By 巧庭_Ann 06 August 2021

先分享自己架網站的動機，

其實一開始只是想找關於Github的教學，

其中就搜尋到Github可以架設免費網站，

就想起:曾經也想過架設一個自己的個人網站，選擇自己喜歡的樣式，分享自己的生活點滴，或是學習過程，

不過很多都是要費用的，每年都需要付年費給主機商，了不起就是免費幾天試用而已，

看到免費架站術這個關鍵字，就吸引我點進去了。

重點重複一次：架設網頁是免費！免費！免費！

↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓

小小筆記一下操作

  1. 第一次到hugo官網挑喜歡的模板並下載

  2. 下載之後，到<資料夾路徑hugo\exampleSite\content\blog>新增文章

(如果不知道怎麼新增文章，先下載自己習慣的原始碼編輯器(例如:Visual Studio Code或PyCharm..等等)就可以新增文章囉)

  1. 打開<資料夾路徑\hugo\exampleSite>的資料夾，顯示路徑的地方全選後，打上cmd

  2. 在cmd上輸入hugo server後，開啟瀏覽器，網址貼上http://localhost:1313/

(能即時看到更新喔~~~ PS.在cmd按下Ctrl+C可以停止)

  1. 停止後，在cmd上輸入hugo

  2. 到<資料夾路徑\hugo\exampleSite\public> 複製所有內容

  3. 創建一個<自己命名.github.io>的資料夾(如:步驟8的資料夾路徑)

  4. 到<資料夾路徑\hugo\u3814520.github.io> 將步驟6所有內容貼上

  5. 到github網頁new repository

(Repository name必須跟步驟7的命名一樣喔)

  1. 電腦沒有git程式，請先下載

  2. 到hugo\u3814520.github.io資料夾右鍵點選git bash，並輸入以下依序輸入步驟10.11.12的內容

  3. git add .

  4. git commit -m “輸入更新內容”

  5. git push origin main

  6. 網址就誕生啦~~~
