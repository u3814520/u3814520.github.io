Linux CentOS網卡消失辦法

By 巧庭_Ann 05 December 2021
Linux CentOS網卡消失不見
  • 發現網卡不見了,只有lo卡
ifconfig
  • 發現了網卡，但是沒有IP地址
ifconfig -a
解決辦法~~~
  1. 查看網卡訊息
sudo lshw -C network
  1. 找到logical name

例如:

logical name: ens33

3.重新獲取IP地址

sudo dhclient ens33

4.發現ens33又出現了，並且具有IP地址

ifconfig
